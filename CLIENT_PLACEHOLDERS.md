# بيانات العميل الجديد — Placeholders

ابحث في المشروع عن العلامات التالية واستبدلها قبل التسليم:

| العلامة | المعنى |
|--------|--------|
| `[[CLIENT_COMPANY_NAME_AR]]` | الاسم التجاري بالعربي |
| `[[CLIENT_COMPANY_NAME_EN]]` | الاسم التجاري بالإنجليزي |
| `[[CLIENT_EMAIL]]` | البريد الرئيسي |
| `[[CLIENT_EMAIL_SALES]]` | بريد المبيعات |
| `[[CLIENT_PHONE]]` | رقم الجوال (عرض محلي) |
| `[[CLIENT_WHATSAPP]]` | واتساب بصيغة دولية بدون + |
| `[[CLIENT_CITY_AR]]` / `[[CLIENT_CITY_EN]]` | المدينة |
| `[[CLIENT_ADDRESS_AR]]` / `[[CLIENT_ADDRESS_EN]]` | العنوان |
| `[[CLIENT_DOMAIN]]` | النطاق بدون https |
| `[[CLIENT_CR_NUMBER]]` | السجل التجاري |
| `[[CLIENT_VAT_NUMBER]]` | الرقم الضريبي |
| `[[CLIENT_PROMO_CODE]]` | مثال كود خصم |
| `[[CLIENT_SNAPCHAT_URL]]` | رابط سناب |
| `[[CLIENT_TIKTOK_URL]]` | رابط تيك توك |
| `[[CLIENT_INSTAGRAM_URL]]` | رابط إنستغرام |
| `[[CLIENT_TWITTER_URL]]` | رابط إكس |
| `[[CLIENT_APPSTORE_URL]]` | رابط App Store |
| `[[CLIENT_PLAYSTORE_URL]]` | رابط Google Play |
