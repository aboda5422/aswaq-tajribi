# تطبيق اسواق تجريبي (Aswaq Tajribi)

قالب متجر بقالة جاهز لإعادة البيع (ويب + تطبيق Capacitor).

## التخصيص للعميل الجديد

1. عدّل `src/config/branding.ts`
2. راجع قائمة العلامات في `CLIENT_PLACEHOLDERS.md` وابحث عن `[[CLIENT_` في المشروع
3. غيّر الألوان في `src/index.css` عند الحاجة
4. حدّث الشعار في `src/assets/logo.png` و `public/favicon.png`

## المعرفات الحالية

- الاسم: تطبيق اسواق تجريبي / Aswaq Tajribi
- Package ID: `com.aswaq.tajribi`
- npm name: `aswaq-tajribi`

## التشغيل

```bash
npm install
npm run dev
```
